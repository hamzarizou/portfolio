# Portfolio — Hamza Aarizou

Site portfolio multi-pages au style futuriste/cyber/hacker. Conçu en HTML/CSS/JS vanilla, prêt pour GitHub Pages.

## Pages
- `index.html`: Accueil (hero glitch, terminal, CTA)
- `about.html`: Présentation + compétences
- `projects.html`: Projets avec liens GitHub
- `internship.html`: Détails de stage (labo de maths)
- `contact.html`: Formulaire (ouvre le client mail via `mailto:`)

## Arborescence
```
css/style.css
js/main.js
assets/images/
```

## Développement local
Ouvrir `index.html` dans votre navigateur. Aucun serveur requis.

## Déploiement sur GitHub Pages
1. Créez un dépôt et poussez tous les fichiers à la racine du dépôt.
2. Sur GitHub, allez dans Settings → Pages.
3. Source: sélectionnez `Deploy from a branch` puis branche `main` et dossier `/ (root)`.
4. Enregistrez. Le site sera publié à l’URL indiquée (quelques minutes d’attente).

## Personnalisation
- Logo/favicon: placez une image dans `assets/images/favicon.png`.
- Couleurs: variables CSS dans `:root` (`css/style.css`).
- Thème clair/sombre: bouton lune dans l’en-tête (préférence stockée en `localStorage`).

## Tech
- HTML5 sémantique, CSS3, JavaScript vanilla.
- Sans framework (pas de Bootstrap/Tailwind/React).
- Responsive mobile/desktop.
